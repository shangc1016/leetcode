



### 输入

直接用cin，忽略控制字符

### 输出

输出保留位数
```cpp
float val = 0.123456;
printf("%.3f", val);
```

四舍五入 
```cpp
float val = 0.123456;
cout << (int)(val +0.5) << endl;
```

向上取整
```cpp
float val = 0.123456;
cout << ceil(val) << endl;
```

向下取整
```cpp
float val = 0.123456;
cout << (int)val << endl;
```

前面补0
```cpp
// cout 用法
int val = 5;
cout.fill('0');         // 设置填充字符
cout.width(5);          // 设置域宽
cout << val << endl;    // 输出


// printf用法
printf("%03d\n", val);
// 0:表示用0填充
// 3:表示宽度为3
int width = 3;
printf("%*d\n", width, val);
```
